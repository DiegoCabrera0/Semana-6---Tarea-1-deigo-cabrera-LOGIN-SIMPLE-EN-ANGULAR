#  Sistema de Login Angular + .NET

##  Descripción
Proyecto de autenticación simple utilizando Angular y .NET 8 Web API.
La autenticación se realiza mediante Cookies HttpOnly seguras.

---

##  Tecnologías utilizadas

- Angular 17+
- .NET 8 Web API
- Autenticación por Cookies
- CORS con AllowCredentials
